# Bidder - The bidding app

A flutter based app to initiate and participate in silent auctions to easily sell off collectibles, antiques, etc.

## Feautures

* Adding Products and Initiating Auctions
* Quick Sell Price
* Silent Bidding
* Resell
* Favourite's Notification
